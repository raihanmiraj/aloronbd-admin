# AloronBD Admin

It is Complete Quiz Admin Project. 

## Installation

Just clone this , and run 

```bash
npm start
```


## Features

- Quiz Timer
- Question/Options Soring 
- Quiz/subject/question add/remove/edit
- Quiz Analytics
